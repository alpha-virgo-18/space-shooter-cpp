# 🚀 Space Shooter Game

A 2D arcade-style **Space Shooter** game developed in **C++** using the **SFML** library as part of an Object-Oriented Programming (OOP) semester project. The game demonstrates core OOP concepts while providing an engaging gameplay experience with enemy waves, boss battles, power-ups, and score tracking.

## ✨ Features

- 🚀 Control a player spaceship with smooth movement
- 👾 Fight multiple enemy waves
- 👑 Face a challenging boss battle
- 🔫 Shooting mechanics with collision detection
- ⚡ Collect power-ups to enhance gameplay
- ❤️ Health and score management
- 🎯 Increasing game difficulty and progression
- 🎵 Integrated game assets including fonts and textures

## 🛠️ Technologies Used

- **Language:** C++
- **Graphics Library:** SFML (Simple and Fast Multimedia Library)
- **IDE:** Visual Studio

## 📚 Object-Oriented Programming Concepts

This project was developed to strengthen understanding of fundamental OOP principles, including:

- Classes & Objects

- Inheritance

## 🚀 Getting Started

### Prerequisites

- Visual Studio
- SFML 2.x
- C++17 compatible compiler

### Installation

1. Clone the repository

```bash
git clone https://github.com/alpha-virgo-18/space-shooter-cpp.git
```

2. Open the project in Visual Studio.

3. Configure SFML libraries and include paths.

4. Build and run the project.

## 🎮 Gameplay

Players control a spaceship and must eliminate incoming enemies while avoiding enemy attacks. As the game progresses, difficulty increases through stronger enemies and a boss encounter. Players can collect power-ups to improve survivability and maximize their score.
